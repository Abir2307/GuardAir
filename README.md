# GuardAir
Real-Time IoT Fault Detection &amp; Air Quality Monitoring System
Technologies: Python, TensorFlow, Scikit-Learn, Streamlit, Plotly, Pandas
•	Developed a real-time IoT monitoring and anomaly detection system for multi-sensor air quality environments. 
•	Implemented hybrid fault-detection models using Isolation Forest, Autoencoders and rule-based validation to identify sensor failures and abnormal environmental conditions. 
•	Built an interactive Streamlit dashboard providing live health monitoring, anomaly visualization, alert management and explainable AI insights. 
•	Designed cyber-attack detection mechanisms for identifying sensor spoofing and saturation attacks through sensor-behaviour analysis. 
•	Automated reporting pipelines with PDF, Excel and CSV exports for operational monitoring and incident analysis.
